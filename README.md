#### 1) What is the difference between var, let, and const?
ans: Difference between var, let, and const:

var: Function-scoped, can be redeclared and updated.

let: Block-scoped, can be updated but not redeclared in the same scope.

const: Block-scoped, cannot be redeclared or updated (constant value).
#### 2) What is the difference between map(), forEach(), and filter()? 
ans:Difference between map(), forEach(), and filter():

map(): Returns a new array with transformed values.

forEach(): Executes a function on each element, does not return a new array.

filter(): Returns a new array with elements that pass a condition.
